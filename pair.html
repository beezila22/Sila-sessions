About SILA-MD

SILA provides a secure pairing flow for your bot or device. This interface is designed to work with a backend endpoint /code?number= but gracefully falls back to a local generator when the server is unavailable.

Quick tips
  • Choose the correct country code from the list.
  • Enter number without spaces or leading zero.
  • Use "Open WhatsApp" to share pairing code quickly.
